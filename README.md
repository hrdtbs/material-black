##  My Custmize


bash_profile
```
$ export PS1="\[\033[36m\]\u\[\033[m\]@:\[\033[33;1m\]\w\[\033[m\]\$ "
$ export CLICOLOR=1
$ export LSCOLORS=ExFxBxDxCxegedabagacad
$ alias ls='ls -GFh'
```
